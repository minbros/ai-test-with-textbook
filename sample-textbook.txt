1. 다항식의 연산
a. 다항식의 덧셈과 뺼셈
{ 다항식의 덧셈은 동류항끼리 모아서 정리하면 된다.
다항식의 덧셈에 관한 성질은 다음과 같다. 교환법칙: A + B = B + A, 결합법칙: (A + B) + C = A + (B + C)
다항식의 뺄셈은 빼는 식의 각 항의 부호를 바꾸어서 더하면 된다. }
b. 다항식의 곱셈과 나눗셈
{ 다항식의 곱셈은 분배법칙을 이용하여 식을 전개한 다음 동류항끼리 모아서 정리한다.
다항식의 곱셈에 관한 성질을 다음과 같다. 교환법칙: AB = BA, 결합법칙: (AB)C = A(BC), 분배법칙: A(B+C) = AB + AC, (A+B)C = AC + BC
다항식의 나눗셈은 각 다항식을 내림차순으로 정리한 다음 자연수의 나눗셈과 같은 방법으로 계산한다.
일반적으로 다항식 A를 B (B ≠ 0)로 나누었을 때의 몫을 Q, 나머지를 R이라 하면, A = BQ + R과 같이 나타낼 수 있다.
이때 R의 차수는 B의 차수보다 낮다. 특히, R = 0, 즉 A = BQ일 때, ‘A는 B로 나누어 떨어진다’고 한다. }

2. 복소수
a. 복소수의 정의
{ 복소수는 실수부와 허수부로 이루어진 수이다.
복소수 z는 z = a + bi로 나타낼 수 있다. 여기서 a는 실수부, b는 허수부, i는 허수단위이다. }
b. 복소수의 연산
{ 복소수의 덧셈과 뺄셈은 실수부와 허수부를 각각 더하거나 빼면 된다.
복소수의 곱셈은 분배법칙을 이용하여 식을 전개한 다음 동류항끼리 모아서 정리한다.
복소수의 나눗셈은 분모와 분자에 공약수를 곱하여 실수부와 허수부를 각각 나누면 된다.
복소수의 제곱근은 다음과 같이 구할 수 있다. z = a + bi일 때, z의 절댓값은 |z| = √(a^2 + b^2)이다.
또한, z의 켤레복소수는 z의 허수부의 부호를 바꾸어서 얻을 수 있다. }
